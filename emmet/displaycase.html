FullGrid

A grid system for you to use

Welcome

FullGrid is made for easy use. Inspired by the Bootstrap grid system. It should offer familiarity and flexibility with a smaller workload.

Use

FullGrid works by using 5 different screen sizes. These are extra large, large, medium, small and extra small. Each screen size is proportionally divided by 12 columns. Any number of columns works as long is it doesn't exceed 12.

Example

We start by creating a container which holds a row with columns. In this example we use the modifier xl-12 to give it a width of 12 colums. Which is the greatest number of columns. The modifiers l, m, s, and xs all indicate the width should be 12 columns across all 5 breakpoints. 

<div class="container">
  <div class="row">

    <div class="column xl-12 l-12 m-12 s-12 xs-12">
     </div>  

  </div>
</div>

!div class=$container$! !div class=$row$! !div class=$column xl-12 l-12 m-12 s-12 xs-12$! !/div! !/div! !/div!

Turns into:
1 big column!
